Emojicode

  • Static, strongly typed language
  • Compiled language
  • It's coded in emoji!

Primitive Types

  • 🚂 - Integer [-2^63+1, 2^63-1]
  • 👌 - Boolean
  • 🚀 - Real Numbers

Honourable mention: s class

  • S Class is the String class
  • 🔡 - String literals
  • 🔡Apples are great🔡 is a string

Emoji Meanings

  • 🍇 - Opening Bracket 🍉 - Closing bracket
  • 🍮 var_name_here - Declare a variable
  • 👴 - Commment this line of code
  • 🍊 - If
  • 🍋 - Else if
  • 🍓 - Else
  • 😀 🔤Hello there🔤 - Print line
  • 🍫 counter_variable - Increment variable
  • 🍳 counter_variable - Decrement variable
  • 🍎 return_variable - Return

Classes

  • 🐇 class - Define a new class, where class must be an emoji
  • 🍰 variable_name type - Declare a class-wide variable. Example: 🍰 counter 🚂
  • 🐈 - Initalise a class Example: 🐈 💳 @flashOn 👌 @megapixels 🚂
  • 🔷 className initializerName [arguments ...] - Instatiate a class Example: 🔷🔡🚂 i 10
  • 🐕 - This, referring to the current object
  • 🏁 - The main function, where the program begins

Installing Emojicode

Link: Install Guide & Download

  • Linux/Windows: Requires installing emoji-font, built into OS X
  • Windows: May be difficult to setup, compiler is similar to C
  • OS X: Install using the install.sh and you're good to go.

Compiling Emojicode

Link: Compiling & Running Guide

  • In terminal, "emojicodec file_name.emojic" will give you file_name.emojib binary
  • To run the binary - "emojicode file_name.emojib"

Coding in Emoji

  • Recommended IDE/Editor: ATOM
  • Atom displays Emoji in the editor, making it easy to work with
  • OS X Tip: ctrl + cmd + spacebar = popup emoji selector
