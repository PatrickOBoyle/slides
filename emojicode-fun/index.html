Installing Emojicode

Link: Install Guide & Download

  • Linux/Windows: Requires installing emoji-font, built into OS X
  • Windows: May be difficult to setup, compiler is similar to C
  • OS X: Install using the install.sh and you're good to go.

Compiling Emojicode

Link: Compiling & Running Guide

  • In terminal, "emojicodec file_name.emojic" will give you file_name.emojib binary
  • To run the binary - "emojicode file_name.emojib"

Coding in Emoji

  • Recommended IDE/Editor: ATOM
  • Atom displays Emoji in the editor, making it easy to work with
  • OS X Tip: ctrl + cmd + spacebar = popup emoji selector
